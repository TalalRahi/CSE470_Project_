# Bangladesh Grassroots Football Talent Tracking System

A MERN stack application for managing grassroots football players.

## Features

✅ User Registration & Login  
✅ Player Profile Management  
✅ Match Tracking  
✅ Analytics & Rankings  
✅ Scouting & Trials  

## Tech Stack

- **Frontend:** React.js
- **Backend:** Node.js + Express.js
- **Database:** MongoDB
- **Authentication:** JWT + bcryptjs

## Installation

### Backend Setup
```bash
cd backend
npm install
```

Create `.env` file:

Start:
```bash
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

## Running Project

Terminal 1:
```bash
cd backend && npm run dev
```

Terminal 2:
```bash
cd frontend && npm start
```

Go to: http://localhost:3000
